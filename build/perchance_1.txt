// The HTML panel's buildFreshnessBlock() calls root.freshSeeds(opts) to draw

// freshSeeds is a sync top-level function with signature:
//   freshSeeds({ profileTags: string[], n: number, excludeSet: string[] }) → string[]
//
// Returns up to n phrases that match at least one profileTag, excluding any
// phrase in excludeSet (the rotation memory of recently-used anchors).
//
// IMPORTANT: corpus is embedded INLINE in the function body — Perchance's parser
// rejects chained list access (treats dot paths through root as list names),
// so we can't keep the bank as a separate list and reach into it. Inlined
// here, only `root.freshSeeds(...)` ever appears in HTML-panel source.
